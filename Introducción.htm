React.js Intruccón

React es una biblioteca o livreria JavaScript automente suficiente y declaratica que se ultiliza para el desarrollo de interfases de usuarios

React es un alibreria que lo unicio que hace es criar interfaces de formas dinamicas

Criando por halla de marzo de por engeñeiro de facebook

Ventajas

Alto rendimiento:React es conocido por su alta eficiencia y flexibilidad. Se puede integral facilmente con diferentes tecnologias. Se puede usar tanto para el lado del cliete cuanto para el lado del Servidor

Recursos abundantes: Como facebook la mantien, existe una gran cantidad de documenteacón y recursos en la web que hace que la curva de apredisaje sea muy fluida

Compatibilidad con versiones anteriores: la transición o migración de versión anteriores a nuevas es bastante fácil y retrocompatibles.

Estructura de componentes facil de mantener:La arquitectura basada en componentes de React ayuda a aumentar la reulización de codigo y facilita bastante el mantenimento del projecto a gran escala.

Fuerte comunidad: React tiene mas de 1300 colaboradores en GIT HUB

Documenteacón multi Idioma: Actualmente React tiene su documenteacón en varios idiomas entre ellos el Español.

Flujo de datos unidireccional: el enlace de datos unidireccional y hacia abajo (de componentes padres y hijos) ayuda garantizar que cambios realizados en el componente hijos no afecte la estructura del componente padre.

Desventajas

Complejo: Muchos desarrolladores pueden encontrar en un inicio demasiado compleja la curva de aprendizaje de React en comparación con otros frameworks como el caso del Vue. Dicha complejidad pueden ser innecessaria para projectos a pequeñas escala.

JSX:el uso de JSX agrega otra capa de complejidad. JSX es un processador que agrega sitaxis XHL a JavaScript. Aunque JSX ayuda a codificar el codigo React de una manera mas segura y rapida, puden ser dificil de comprender para los nuevos desarrolladores.

Necesidad de um ecosistema de muchas herramientas: React requiere una amplia gama de herramientas para funcionar correctamente y ser complatible con otras tecnologias.

Problemas de SEO: Se sabe que las SPAs(Single Page Applications) creadas com React se enfrentan a probelams de indexación por parte de los rastreadores y bots de motores de busquedas.

¿Quién lo usa?

Grandes empresas como Netflix, Yahoo, Airbnb, intagram, facebook, WhatsApp, PayPal, Microsoft, la BCC, Aeromexico,etc. Incluso grandes sitios de informacion y noticias que antes usaban WordPress ser estan migrando a sitios hechos con JAM stack y React como libreria principal, tal es ek caso de smarhing magazine.

Entorno y herramientas de Desarrollo

Node.js y NPM Para ejecutar el entorno de desarrollo e instalar dependecias

Yarn Gestor de paquetes JavaScript

WebpackEs un empaquetador de archivos para JavaScript

React Router Libreria para manejar rutas declarativas (url)

ReduxEs una libreria para gestinar el estado de la aplicaciones en javaScript

FluxEs la arquitectura de aplicaciones que facebook usa para crear aplicaciones web del lado del cliete

Create React App: Un conjunto de configuraciones preestabelecidas para comenzar a trabajar con React ejecutando un solo comando

Gatsby: Es un framework basado en React para desarollar rapidamente sitios y aplicaciones web, usando diferentes fuentes de datos como CMS, Markdowns,APIs, etc.

Next.js: Es un framework basado en React para desarollar sitios estaticos y aplaciones del lado del servidor(SSR - Server Side Rendering)

GraphQL: Es una linguaje de cosulta de datos para APIs.

React Bootstrap: Es la version del popular Framework frontend Bootstrap pero criado con componentes hechos en React

Material UI: Es un framework frontend inspirado en material Design creados con componentes hechos en React.
